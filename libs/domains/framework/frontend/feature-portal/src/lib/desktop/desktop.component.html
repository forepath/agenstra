Full console, native experience

The same powerful Agenstra console you use in the browser - packaged as a standalone desktop application for Windows, macOS, and Linux.

Native install

Download and install like any desktop app. No browser tabs, no web URLs to remember. Launch Agenstra from your dock, taskbar, or application menu - Windows (.zip), macOS (.zip), or Linux (.deb, .zip).

Full console inside

The complete Agenstra console: Monaco-based IDE, real-time chat with AI agents, file tree, terminal, Git integration, and container management. Everything you get in the web version - in a dedicated window.

Cross-platform

Built with Electron for consistent behavior across platforms. Use the same config and workflows whether you're on Windows, macOS, or Linux. Connect to remote agent-manager instances from anywhere.

Hands-free enterprise deployment

Point the desktop app to a central config URL. No rebuilds, no reinstalls - ideal for enterprise deployments.

Single source of truth

Host one JSON config file on your internal server. Every client fetches it at launch. Update the file once and all deployments pick up the change automatically - no per-machine edits.

Scale without friction

Roll out the same installer across the entire organization. Users point to the appropriate config URL for their environment. No custom builds or per-site packaging.

Change config, not the app

Switch backends, update auth, or toggle features by editing the remote config. No rebuilds, no redistributing. The same pattern works for Docker via the CONFIG environment variable.
